# lost-and-found
## Project setup
```
npm install
```

### Navigate 
```
 cd yolo_api_local
```
### Create venv 
```
python -m venv venv
```
### then activate 
```
venv\Scripts\activate
```

```
pip install -r requirements.txt
```

```
npm run dev
```

```
npm run build
```
